# Hybrid SaaS Deployment

**Status:** Accepted  
**Date:** 2026-07-07  
**Deciders:** Architecture Review Board

## Context

Deployment model ambiguity (AMB-001)

## Decision

Hybrid SaaS default with dedicated enterprise tier and air-gapped profile option.

## Consequences

+ Scale economics, enterprise SKUs. - Multiple profiles to test.
