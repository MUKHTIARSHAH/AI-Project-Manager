# Canonical Constitution Path

**Status:** Proposed  
**Date:** 2026-07-07  
**Deciders:** Architecture Review Board (pending)

## Context

PC-AIPM-001 was authored at `00-Governance/Project-Constitution.md` as the supreme engineering authority. The Engineering Blueprint README references folder `03-Project-Constitution/` for constitutional law. Two locations create discovery risk.

## Decision

**Proposed:** Canonical path for PC-AIPM-001 is `00-Governance/Project-Constitution.md`. Folder `03-Project-Constitution/` is reserved for supplements only and contains a redirect stub.

## Consequences

+ Single source of truth alongside Vision, Charter, and Glossary.  
+ README folder `03` may be renamed or repurposed in a future governance update (requires Change-Log, does not modify locked SRS/SAD).  
- Minor inconsistency with README v1.1.0 until README is updated through governance process.
